Description
Main DescriptionThis model describes the software architecture, which depends on the solution domain. For example, the architecture of a service composition can be represented by a SoaML architecture model, or the architecture of a component-based system can be described by an architecture description language (ADL). Our framework is flexible on the notation used to describe this model.
