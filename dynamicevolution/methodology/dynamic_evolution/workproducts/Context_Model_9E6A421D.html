Purpose

This artifact represents an ontology-based context model. It leverages Semantic Web technology to solve the need for expressing the computing infrastructure in a way that supports formal reasoning of its current status and possible arising situations.

Description
Main Description

The context model makes use of the Web Ontology Language (OWL) to support the formal analysis of the context information that is taken from the computing infrastructure. OWL extends the expressivity of the Resource Description Framework (RDF) by adding an additional layer of semantics on top of RDF. With RDF, the contextual knowledge can be decomposed into small pieces, with some rules about the meaning (or semantics) of those pieces.
