Artifact: Tactic Model
This artifact represents a tactic model that describes a tactic.
Purpose
The set of tactic models describe surviving tactics to preserve the requirements at runtime.

Description
Main DescriptionTactic models are causally connected to software that implement tactics. Also, tactic models are merged into the variability model at runtime to include the tactical functionality in the evolved system configuration. The only merging prerequisite is that these two models conform to the same metamodel. Each tactic model can have restrictions to indicate if the tactic can run in parallel with other current active system functionalities, or if it is necessary to activate or deactivate current functionalities to apply the tactic. In addition, tactic models are expressed as general as possible to be reused.
