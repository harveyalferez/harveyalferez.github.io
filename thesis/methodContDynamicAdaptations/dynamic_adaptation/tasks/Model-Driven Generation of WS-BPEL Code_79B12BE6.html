Task: Model-Driven Generation of WS-BPEL Code
Purpose
Model-driven generation of WS-BPEL code.

    Outputs
    • WS-BPEL Code Fragments
    • WS-BPEL template
      Main Description
      The objective of this task is to generate the WS-BPEL code that is going to be used during dynamic adjustments. Specifically, a WS-BPEL template is created to be instantiated at runtime with a set of WS-BPEL code fragments, which represent variants. These fragments are stored in a repository. The generation of the WS-BPEL template and the WS-BPEL code fragments is model-driven.
