Task: Implement the Service Operations for Tactics
Purpose
Implement the service operations for tactics.
Relationships
RolesPrimary Performer:
  • Developer
Additional Performers:
Outputs
  • Atomic Web Services
  • Composite Web Services
    Main Description
    The objective of this task is to implement the service operations for each tactic. Some tactics can be implemented as atomic Web services (their implementation is self contained and does not invoke any other services). Other tactics can be implemented as composite Web services. These tactics are more complex than the ones implemented as atomic Web services because they can invoke several service operations.
