Description
Main DescriptionThe weaving model reflects the activation of features in tactic models, which are expressed as feature models, on composition models that abstract the workflow among service operations in these tactics. Each link in this weaving model has the following endpoints: the first endpoint refers to features in all the tactic models that are expressed as feature models; the second endpoint refers to elements in all the tactic models that are expressed as composition models.
